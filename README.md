# Astro Party - Многопользовательская космическая игра

Многопользовательская игра в стиле Astro Party, где игроки управляют космическими кораблями через WebSocket с любого устройства.

## Особенности

- До 10 игроков одновременно
- Управление с любого устройства (мобильные и ПК) через WebSocket
- Поддержка клавиатуры (ПК) и сенсорного управления (мобильные)
- Автоматическое движение кораблей с инерцией
- Реальное время синхронизация
- Красивая космическая графика

## Установка

1. Установите зависимости:

```bash
npm install
```

2. Запустите сервер:

```bash
npm start
```

Или для разработки с hot-reload (автоматическая перезагрузка при изменении файлов):

```bash
npm run dev
```

**Предотвращение спящего режима (macOS):**

Если компьютер уходит в сон во время игры, используйте скрипты с `caffeinate`:

```bash
npm run start:no-sleep
```

Или для разработки:

```bash
npm run dev:no-sleep
```

Эти команды предотвратят переход компьютера в спящий режим во время работы сервера.

Сервер запустится на порту 3000 (или на порту, указанном в переменной окружения PORT).

## Использование

1. **Главный экран (отображение игры):**

   - Откройте в браузере: `http://localhost:3000`
   - Или: `http://localhost:3000/index.html`

2. **Устройство управления:**
   - Откройте на любом устройстве: `http://[IP-адрес-сервера]:3000/control.html`
   - Например: `http://192.168.1.100:3000/control.html`
   - Убедитесь, что устройство находится в той же сети, что и сервер

## Управление

### На ПК (клавиатура):

- **Пробел**: Поворачивает корабль направо (удерживайте для непрерывного поворота)
- **Enter**: Выпускает снаряд

### На мобильных устройствах (сенсорное):

- **Левая кнопка (ПОВОРОТ)**: Поворачивает корабль направо
- **Правая кнопка (ВЫСТРЕЛ)**: Выпускает снаряд

Корабль автоматически движется вперед. Цель игры - уничтожить всех остальных игроков.
